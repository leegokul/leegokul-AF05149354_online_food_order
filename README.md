# 🍔 Online Food Order System

An Online Food Order System developed using Java Full Stack technologies with JDBC, Servlets, MySQL, HTML, and CSS.

This project allows customers to register, log in, view restaurants, browse food menus, place orders, and view their order history.

---

## 🚀 Features

- Customer Registration
- Customer Login
- Session Management
- Logout System
- View Restaurants
- Restaurant-wise Food Menu
- Place Food Orders
- View Order History
- Video Background UI
- Image Background UI
- Responsive Frontend Design

## ▶ How to Run the Project

1. Import the project into IntelliJ IDEA
2. Configure Apache Tomcat
3. Add MySQL Connector JAR
4. Create the MySQL database and tables
5. Run the Tomcat server
6. Open browser and run:

```text
http://localhost:8080
```
